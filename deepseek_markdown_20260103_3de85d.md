# كوكي كيكر المتطور - لعبة متعددة اللاعبين

## المميزات
- نظام تطويرات متقدم
- لوحة متصدرين عالمية
- إمكانية اللعب مع لاعبين حقيقيين
- تصميم عربي متجاوب
- حفظ تقدم اللعبة محلياً

## التقنيات المستخدمة
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: WebSockets
